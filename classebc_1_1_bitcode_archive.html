std::vector< BitcodeFile > ebc::BitcodeArchive::GetBitcodeFiles ( ) const
overridevirtual

Extract individual bitcode files from this archive and return a vector of file names. This operation can be expensive as it decompresses each bitcode file. The result is empty if not compiled with xar support.

Reimplemented from ebc::BitcodeContainer.

const BitcodeMetadata & ebc::BitcodeArchive::GetMetadata ( ) const

Return the MetaData contained in this bitcode archive. This operation is cheap as the heavy lifting occurs at construction time. Metadata is empty if not compiled with xar support.

std::string ebc::BitcodeArchive::WriteXarToFile ( std::string  fileName = "") const

Write container data to file. If no file name is provided, the file format name of the binary will be used, followed by the xar extension. This works even when compiled without xar support.
